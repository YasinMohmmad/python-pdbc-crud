# Python PDBC CRUD Mini Project

## Student Management System using Python + MySQL

A mini project developed using **Python Database Connectivity (PDBC)** and **MySQL** to perform complete **CRUD Operations** through a menu-driven console application.

---

## Project Overview

This application allows users to interact with a MySQL database and perform:

* Insert Student Records
* Select and View Records
* Update Existing Data
* Delete Records
* Exit Application

The project also includes exception handling to manage runtime and database-related errors.

---

## Tech Stack

* Python
* MySQL

* VS Code
* Git & GitHub

---

## Features

### Create (Insert)

Add student details into the database.

### Read (Select)

Retrieve and display stored records.

### Update

Modify existing student information.

### Delete

Remove records from the database.

### Exception Handling

Handles:

* Duplicate Primary Key Errors
* Invalid Input Errors
* Runtime Exceptions

### Database Connectivity

Uses:

* `connect()`
* `cursor()`
* `execute()`
* `commit()`
* `fetchall()`
